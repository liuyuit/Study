想要排除 service 下的所有文件，但是排除 `service/dockerfile`

可以在 根目录下的 `.gitignore`  加入

```
#service/*
#!service/dockerfile
```

或者

` vim service/.gitignore`

```
/*
!/dockerfile
!/.gitignore
```
